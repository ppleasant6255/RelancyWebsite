Secrets Overview

Secrets such as API keys, OAuth tokens, and integration credentials are stored in a secure secrets vault. Secrets are encrypted, access‑controlled, and never exposed in logs or exports.

Storage rules

  • Encrypted at rest using tenant‑scoped keys
  • Access restricted to authorized system components
  • No plaintext storage anywhere in the platform

Rotation policies

Secrets may be rotated manually by administrators or automatically based on expiration or security events.

Audit integrity

All secret creation, rotation, and access events are logged for compliance and operational review.
