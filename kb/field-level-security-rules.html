Security Overview

Field‑level security ensures that sensitive information is visible and editable only to authorized roles. Security rules apply across incidents, cases, investigations, training, readiness, and reporting.

Security controls

  • View restrictions
  • Edit restrictions
  • Protected‑class field masking
  • Confidential note visibility
  • Role‑based field overrides

Operational impact

Field‑level security ensures compliance with privacy regulations and internal confidentiality policies.

Audit integrity

All field access and edits are logged with timestamp, actor, and justification when required.
