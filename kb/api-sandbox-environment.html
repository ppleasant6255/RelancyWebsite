Sandbox Overview

The API Sandbox Environment allows developers to test integrations using mock data, simulated workflows, and isolated tenant environments. Sandbox usage does not affect production systems.

Sandbox capabilities

  • Mock incidents, cases, and training data
  • Simulated webhook delivery
  • API key generation for sandbox only
  • Rate‑limited testing environment

Operational safeguards

Sandbox environments cannot access production data or systems. All sandbox keys and endpoints are isolated and expire automatically.

Audit integrity

Sandbox activity is logged separately from production for troubleshooting and developer support.
