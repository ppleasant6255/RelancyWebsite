Dashboard Overview

Dashboards provide high‑level visibility into incidents, cases, readiness, and risk. Each widget summarizes key metrics and supports drill‑down navigation for deeper analysis.

Widget types

  • Incident volume charts
  • Case severity distribution
  • Readiness scores
  • Training compliance
  • Location risk indicators
  • Corrective action status

Drill‑down behavior

Tapping a widget opens a filtered list or detailed report showing the underlying records. Filters are preserved as users navigate deeper.

Role‑based visibility

Dashboard content is tailored to the user’s role, ensuring appropriate access to sensitive data.

Export options

Users may export widget data for offline analysis or leadership reporting.
