Search Overview

Relancy includes a unified search engine that supports keyword search, structured filters, and entity‑specific queries. Search results are tenant‑scoped and respect RBAC permissions.

Supported entities

  • Cases
  • Incidents
  • Investigations
  • Employees
  • Training records
  • Locations
  • Departments

Filtering options

  • Category and severity
  • Status and lifecycle stage
  • Date ranges
  • Location or department
  • Assigned owner
  • Tags and metadata

Search performance

Search is optimized for speed and returns results in real time, even for large tenants.

RBAC enforcement

Users only see results for entities they are authorized to view.
