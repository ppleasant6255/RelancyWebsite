Maintenance Overview

Relancy performs scheduled maintenance to apply updates, security patches, and infrastructure improvements. Maintenance windows are designed to minimize tenant impact and maintain platform reliability.

Maintenance windows

  • Scheduled during low‑usage periods
  • Announced in advance via email and in‑app notifications
  • Critical fixes may be applied outside scheduled windows
  • Private Deployment tenants may define custom windows

Release cadence

  • Monthly feature releases
  • Bi‑weekly bug fix updates
  • Continuous security patching
  • Feature flags for controlled rollout

Operational impact

Most updates occur without downtime. When downtime is required, Relancy provides clear communication and post‑maintenance summaries.
