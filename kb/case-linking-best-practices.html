Case Linking Best Practices

How to link related cases, incidents, and investigations effectively and defensibly.

Back to Knowledge Base

Overview

Case linking helps organizations understand relationships between incidents, investigations, and operational patterns. Proper linking improves reporting accuracy and supports root cause analysis.

When to link cases

  • Multiple incidents involving the same individual
  • Repeat behavior or escalating patterns
  • Cross-department or cross-location events
  • Cases sharing evidence or witnesses
  • Follow-up cases related to corrective actions

Linking guidelines

  • Link only when a meaningful relationship exists
  • Document the reason for linking in case notes
  • Avoid over-linking unrelated cases
  • Review links during case closure

Reporting benefits

  • Improved pattern detection
  • Better risk analysis
  • Clearer operational narratives
  • Enhanced compliance documentation

Audit logging

All linking and unlinking actions are logged in the audit trail.
