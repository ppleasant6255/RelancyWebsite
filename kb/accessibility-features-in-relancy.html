Accessibility Overview

Relancy is designed to meet accessibility standards and support users with diverse needs. Accessibility features ensure that all employees can navigate, read, and interact with the platform effectively.

Supported features

  • High‑contrast mode
  • Screen reader compatibility
  • Keyboard navigation support
  • Accessible form labels and controls
  • Scalable text sizes

Compliance alignment

Relancy aligns with WCAG accessibility guidelines and industry best practices.

User preferences

Accessibility settings can be adjusted in user preferences and persist across sessions.
