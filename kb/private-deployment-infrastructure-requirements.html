Private Deployment Infrastructure Requirements

Infrastructure and configuration requirements for hosting Relancy in a dedicated environment.

Back to Knowledge Base

Overview

Private Deployment provides dedicated hosting for organizations requiring enhanced control, isolation, or compliance. This article outlines infrastructure requirements for a successful deployment.

Compute requirements

  • Dedicated application servers
  • Scalable compute clusters
  • High‑availability configuration

Storage requirements

  • Encrypted storage volumes
  • Tenant‑isolated containers
  • Backup and retention policies

Networking requirements

  • Secure inbound and outbound routing
  • Firewall and access control rules
  • TLS termination

Operational requirements

  • Monitoring and alerting systems
  • Disaster recovery planning
  • Patch and update workflows

Support considerations

Private Deployment tenants may require dedicated support agreements or integration assistance.
