Explorer Overview

The Admin Audit Log Explorer provides administrators with visibility into all operational activity, including case actions, evidence handling, configuration changes, and authentication events.

Filter options

  • User or role
  • Entity type
  • Action type
  • Date range
  • Severity or impact level

Use cases

  • Investigating unusual activity
  • Supporting compliance audits
  • Reviewing administrative changes
  • Validating lifecycle exceptions

Export options

Administrators may export filtered audit logs for compliance review or external analysis. Sensitive fields may be redacted.

Security safeguards

Audit logs are immutable and cannot be altered or deleted by tenant administrators.
