Dashboard Overview

The System Health Dashboard provides real‑time visibility into platform performance, uptime, background job throughput, and integration status. It supports proactive monitoring and operational readiness.

Key indicators

  • API response times
  • Background job queue depth
  • Notification delivery rates
  • Webhook success/failure rates
  • Database performance metrics
  • Integration health checks

Operational impact

System health insights help administrators identify issues early, plan capacity, and maintain platform reliability across tenants.

Audit integrity

Health events and anomalies are logged for operational review and compliance alignment.
