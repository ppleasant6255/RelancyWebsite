Framework Overview

The Data Import Framework enables administrators to import structured data into Relancy using validated, secure, and auditable workflows. Imports support HRIS sync, historical migration, and tenant onboarding.

Supported import types

  • Employee records
  • Location and department structures
  • Historical incidents and cases
  • Training completions
  • Certification records

Validation rules

  • Schema validation for required fields
  • Cross‑module consistency checks
  • Duplicate detection
  • Role and reporting line validation

Audit integrity

All imports are logged with timestamp, actor, file metadata, and validation results for compliance and historical traceability.
