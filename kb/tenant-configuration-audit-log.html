Audit Log Overview

The Tenant Configuration Audit Log records all administrative changes to tenant settings, ensuring transparency, accountability, and compliance alignment.

Tracked configuration changes

  • Role and permission updates
  • Workflow configuration changes
  • Integration settings
  • Notification templates
  • Retention and purge policies
  • Branding and customization settings

Operational impact

Configuration audit logs help administrators troubleshoot issues, validate changes, and maintain consistent tenant behavior.

Audit integrity

All configuration changes are logged with timestamp, actor, old value, new value, and justification when required.
