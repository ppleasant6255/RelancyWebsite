Extension Overview

Relancy provides controlled extension points that allow tenants to customize workflows, data structures, and contextual guidance without compromising platform integrity.

Supported extension points

  • Custom fields
  • HTML contextual panels
  • API integrations
  • Webhook listeners
  • Tenant branding

Governance

All extensions must follow platform safety rules, including HTML sanitization, RBAC enforcement, and audit logging.

Private Deployment

Private Deployment tenants may enable additional extension points within their isolated environment.
