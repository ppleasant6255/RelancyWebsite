How Relancy safely renders tenant‑provided HTML panels without exposing cross‑tenant or security risks.

Safety Overview

HTML panels allow tenants to display custom content within Relancy. Safety rules ensure that HTML is sanitized, isolated, and free from scripts or unsafe elements.

Allowed content

  • Formatted text
  • Images (tenant‑scoped)
  • Links (open in new tab)
  • Tables and lists

Blocked content

  • JavaScript
  • Inline event handlers
  • External scripts or iframes
  • Cross‑tenant references

Operational safeguards

HTML is sanitized server‑side and rendered in a sandboxed container to prevent cross‑site scripting or data leakage.

Audit integrity

HTML panel updates are logged with timestamp, actor, and sanitized output for compliance review.
