Telemetry Overview

Platform telemetry provides real‑time insights into system performance, reliability, and usage patterns. Telemetry supports proactive monitoring and capacity planning.

Collected metrics

  • API performance
  • Background job throughput
  • Notification delivery rates
  • User activity patterns
  • Error and exception rates

Privacy safeguards

Telemetry never includes case content, personal notes, or sensitive HR/Security data. Only operational metadata is collected.

Operational impact

Telemetry enables proactive scaling, issue detection, and performance optimization.

Audit integrity

Telemetry logs are retained according to tenant and regulatory requirements.
