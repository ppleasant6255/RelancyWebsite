Module Overview

The Investigation module provides structured workflows for conducting, documenting, and closing investigations. It integrates with evidence management, interviews, and root cause analysis.

Core capabilities

  • Investigation lifecycle
  • Interview management
  • Evidence capture and chain‑of‑custody
  • Root cause analysis tools
  • Findings and recommendations

Access boundaries

Investigations are restricted to authorized HR or Security roles depending on case type.

Audit integrity

Every investigation action is logged, including evidence handling, interviews, and findings.
