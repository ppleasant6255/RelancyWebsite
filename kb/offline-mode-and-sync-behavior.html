Offline Mode and Sync Behavior

How Relancy handles offline access, local caching, and synchronization when connectivity is restored.

Back to Knowledge Base

Offline Overview

Relancy supports limited offline functionality to ensure continuity in low‑connectivity environments. Users may view cached data and create drafts that sync when connectivity returns.

Available offline features

  • View recently accessed cases and incidents
  • Create draft notes
  • Capture evidence for later upload
  • Fill out incident forms offline

Sync behavior

When connectivity is restored, Relancy automatically syncs drafts, uploads evidence, and updates local caches. Conflicts are resolved using server‑side rules.

Limitations

Sensitive or restricted data may not be available offline due to security requirements.
