Catalog Overview

Relancy emits webhook events for key lifecycle changes across incidents, cases, investigations, training, and readiness. Webhooks support real‑time integrations and automation.

Event categories

  • Incident events (created, updated, closed)
  • Case events (assigned, escalated, closed)
  • Investigation events (stage changes, findings)
  • Training events (assigned, completed, expired)
  • Readiness events (score changes)
  • Corrective action events (created, overdue, verified)

Payload structure

Each event includes entity identifiers, timestamps, triggering conditions, and tenant context. Sensitive fields are omitted based on RBAC.

Delivery guarantees

Webhooks use at‑least‑once delivery with retry logic and dead‑letter logging for unresolved failures.
