Employee Status Change Workflow

How Relancy processes employee status changes using HRIS sync, validation rules, and automated updates.

Back to Knowledge Base

Workflow Overview

Employee status changes—such as hire, transfer, leave, or termination—are processed automatically through HRIS sync. Status changes update routing, visibility, and readiness.

Status types

  • New hire
  • Internal transfer
  • Leave of absence
  • Return from leave
  • Termination

Operational safeguards

Terminated employees lose access immediately. Open tasks are reassigned. Historical records remain intact.

Audit integrity

Status changes include timestamp, source system, old value, new value, and validation results for compliance alignment.
