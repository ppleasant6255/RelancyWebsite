Integration API Overview

Overview

Relancy provides secure, read‑only APIs that allow external systems to retrieve operational data for reporting, analytics, and workflow automation. APIs follow strict authentication and rate‑limit rules.

Supported API areas

  • Cases and incidents
  • Investigations
  • Training lineage
  • Readiness data
  • Organizational structure

Authentication

All API requests require secure token‑based authentication. Tokens are tenant‑scoped and time‑limited.

Use cases

  • BI dashboards
  • HRIS synchronization
  • LMS reporting
  • Compliance exports

Private Deployment

Private Deployment tenants may enable extended API capabilities or custom endpoints.
