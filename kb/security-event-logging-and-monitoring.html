Security Event Logging and Monitoring

How Relancy tracks and monitors security‑related activity across the platform.

Back to Knowledge Base

Overview

Security event logging provides visibility into authentication, authorization, and system‑level activity. Relancy monitors these events to detect anomalies and support compliance.

Logged events

  • Login attempts
  • Permission changes
  • API authentication failures
  • Data export activity
  • Webhook delivery failures

Monitoring tools

Relancy uses automated monitoring to detect suspicious activity and notify administrators when necessary.

Compliance alignment

Security logs support audits, investigations, and regulatory reporting.

Private Deployment

Private Deployment tenants may integrate logs with SIEM or external monitoring systems.
