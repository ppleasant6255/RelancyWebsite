Session Overview

Session management ensures secure access, prevents unauthorized use, and supports compliance requirements. Session rules vary by environment and role sensitivity.

Timeout rules

  • Standard timeout for employees and managers
  • Shortened timeout for HR and Security roles
  • Immediate timeout for kiosk mode
  • Configurable tenant‑wide timeout policies

Session safeguards

  • Device‑bound session tokens
  • Automatic logout on role change
  • Forced logout on deprovisioning
  • Session revocation on security events

Audit integrity

All session events—including logins, logouts, timeouts, and revocations—are logged for compliance and security review.
