Governance Overview

Custom fields allow tenants to extend Relancy’s data model while preserving platform integrity. Governance rules ensure that custom fields remain safe, consistent, and compliant.

Supported field types

  • Text and long text
  • Dropdown lists
  • Dates
  • Numeric fields
  • Boolean fields

Governance rules

  • Custom fields cannot override core fields
  • Custom fields must be assigned to specific entities
  • Custom fields follow RBAC visibility rules
  • Custom fields appear in exports only when allowed by policy

Audit logging

Creation, modification, and deletion of custom fields are logged for compliance and historical accuracy.
