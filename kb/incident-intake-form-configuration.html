Configuration Overview

Administrators may configure incident intake forms to align with organizational policy. Configuration includes required fields, category‑specific questions, and routing rules.

Configurable elements

  • Required fields
  • Category‑specific prompts
  • Location selection rules
  • Attachment requirements
  • Confidentiality indicators

Validation rules

Intake forms enforce validation to ensure complete and accurate submissions. Errors must be corrected before submission.

Routing integration

Form configuration directly influences routing paths, severity scoring, and escalation triggers.

Audit logging

All configuration changes are logged for compliance and historical traceability.
