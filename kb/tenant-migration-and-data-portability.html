Tenant Migration and Data Portability

How Relancy supports tenant migration, data export, and portability across environments.

Back to Knowledge Base

Overview

Relancy supports tenant migration and data portability to help organizations transition between environments, regions, or deployment models while preserving audit integrity.

Migration scenarios

  • SaaS → Private Deployment
  • Private Deployment → SaaS
  • Region‑to‑region migration
  • Tenant consolidation or separation

Data export capabilities

  • Cases and incidents
  • Investigations
  • Training lineage
  • Readiness data
  • Organizational structure
  • Audit logs

Integrity safeguards

All exported data preserves timestamps, identifiers, and audit relationships to ensure continuity.

Private Deployment

Private Deployment tenants may perform full‑environment migrations with dedicated support.
