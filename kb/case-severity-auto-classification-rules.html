Auto‑Classification Overview

Relancy uses structured rules to automatically assign severity levels to cases. Auto‑classification ensures consistency, reduces user error, and supports defensible decision‑making.

Severity factors

  • Case category and subcategory
  • Protected‑class involvement
  • Repeat behavior indicators
  • Location risk profile
  • Policy‑defined severity triggers

Override rules

Authorized HR or Security roles may override severity when appropriate. Overrides require justification and are fully audited.

Audit integrity

All auto‑classification events and overrides are logged with contributing factors for transparency and defensibility.
