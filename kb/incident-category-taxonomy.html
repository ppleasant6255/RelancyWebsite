Incident Category Taxonomy

How Relancy organizes incident types into a structured, consistent taxonomy.

Back to Knowledge Base

Overview

A consistent taxonomy ensures accurate reporting, trend analysis, and risk detection. Relancy provides a structured set of incident categories aligned with HR, Safety, and Security workflows.

Core categories

  • Safety
  • Conduct
  • Harassment
  • Security
  • Facilities
  • Policy violation
  • Environmental
  • Operational error

Subcategories

Each category may include subcategories for more granular classification. Subcategories support deeper analytics and pattern detection.

Governance

Administrators manage taxonomy updates. Changes are logged in the audit trail to preserve historical accuracy.

Reporting benefits

  • Improved trend analysis
  • Better risk identification
  • Consistent case classification
  • Enhanced compliance reporting
