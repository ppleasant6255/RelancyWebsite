Penetration Testing Program

Program Overview

Relancy performs regular penetration tests using internal security teams and third‑party specialists. Testing covers application, infrastructure, and integration surfaces.

Testing scope

  • API endpoints
  • Authentication and SSO flows
  • Role‑based access controls
  • Data storage and encryption
  • Integration surfaces

Remediation process

Findings are triaged, prioritized, and remediated according to severity. Critical issues receive immediate attention and verification.

Audit integrity

Penetration test reports and remediation logs are retained for compliance and security review.
