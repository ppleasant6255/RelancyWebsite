Engine Overview

The HRIS Sync Engine integrates with external HRIS platforms to maintain accurate employee, location, and organizational data. Syncs occur on a scheduled basis or via webhook triggers.

Sync capabilities

  • Employee creation and updates
  • Role and department changes
  • Location assignments
  • Manager reporting line updates
  • Employment status changes

Conflict handling

HRIS data is authoritative for core employee attributes. Relancy resolves conflicts using deterministic rules and logs all overrides.

Audit integrity

All sync events—including changes, failures, and overrides—are logged for compliance and operational review.
