Kiosk Overview

When deployed in CAD‑managed kiosk mode, Relancy operates with restricted navigation, controlled session behavior, and enhanced security to support shared‑device environments.

Behavioral rules

  • Restricted navigation to approved workflows
  • Automatic session timeout
  • Blocked external links (open in default browser only)
  • No local data storage beyond safe caches
  • Limited settings access

Operational safeguards

Kiosk mode prevents cross‑user data exposure, enforces logout between sessions, and disables sensitive features such as offline mode or unrestricted evidence capture.

Audit integrity

All kiosk sessions, logins, and restricted actions are logged for compliance and operational review.
