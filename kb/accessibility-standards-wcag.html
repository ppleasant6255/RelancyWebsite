Accessibility Overview

Relancy adheres to WCAG 2.1 AA accessibility standards to ensure that all users—including those with disabilities—can access and use the platform effectively.

Key accessibility features

  • Keyboard‑only navigation
  • Screen reader compatibility
  • High‑contrast color options
  • Accessible form labels and error messages
  • Semantic HTML structure

Compliance alignment

Accessibility standards support compliance with ADA, Section 508, and international accessibility regulations.

Continuous improvement

Accessibility audits are conducted regularly, and user feedback informs ongoing improvements.
