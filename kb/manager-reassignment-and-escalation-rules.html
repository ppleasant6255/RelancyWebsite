Manager Reassignment and Escalation Rules

How Relancy handles manager changes and ensures proper escalation routing.

Back to Knowledge Base

Overview

Manager changes affect case visibility, escalation routing, and readiness oversight. Relancy updates these relationships automatically to maintain operational accuracy.

Reassignment triggers

  • Manager promotion or departure
  • Department restructuring
  • Employee transfer
  • Leadership‑initiated changes

Escalation updates

Relancy updates escalation paths to ensure cases and incidents route to the correct manager or leader.

Audit logging

All manager changes are logged with timestamp, actor, and justification.

Reporting impact

Manager changes update readiness dashboards, case visibility, and department‑level reporting.
