Integration Overview

Relancy integrates with identity providers such as Azure AD, Okta, and Ping to support SSO, MFA, and automated user provisioning. Identity integration ensures secure, centralized authentication.

Supported protocols

  • OpenID Connect
  • OAuth 2.0
  • SAML 2.0 (optional)

Provisioning capabilities

  • Just‑in‑time user provisioning
  • Role and group mapping
  • Automatic deprovisioning
  • Attribute synchronization

Operational safeguards

Identity provider failures trigger fallback authentication rules and administrative alerts.

Audit integrity

All authentication and provisioning events are logged for compliance and security review.
