Injection Overview

Relancy supports safe HTML injection at predefined locations throughout the app. These panels allow tenants to provide contextual guidance, policy reminders, or branding without modifying platform behavior.

Supported injection points

  • Dashboard header
  • Incident intake screen
  • Case overview screen
  • Training overview screen
  • Employee profile header

Safety restrictions

Only a safe subset of HTML and CSS is allowed. Scripts, iframes, external resources, and dynamic content are blocked to maintain security and tenant isolation.

Governance

Administrators manage HTML panels. All changes are logged and versioned for audit purposes.
