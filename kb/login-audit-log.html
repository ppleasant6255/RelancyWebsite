Login Audit Log

How Relancy logs authentication events for compliance, security monitoring, and anomaly detection.

Back to Knowledge Base

Log Overview

The Login Audit Log records all authentication attempts, including successful logins, failures, MFA challenges, and SSO events.

Logged metadata

  • Timestamp
  • Identity provider
  • Authentication method
  • Success or failure status
  • Session metadata
  • Security alerts (if applicable)

Operational safeguards

Failed login attempts trigger lockouts and alerts. Suspicious patterns escalate to security administrators.

Audit integrity

Logs are immutable and retained according to tenant retention and regulatory requirements.
