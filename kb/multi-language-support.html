Language Overview

Multi‑language support allows users to select their preferred language for the Relancy interface. Language settings apply across all modules and workflows.

Supported features

  • User‑selectable language preferences
  • Tenant‑wide default language
  • Localized UI text and messages
  • Fallback language rules

Operational impact

Multi‑language support improves usability for global teams and ensures consistent communication across regions.

Audit integrity

Language changes and localization overrides are logged for compliance and historical review.
