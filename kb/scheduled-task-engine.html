Engine Overview

The Scheduled Task Engine runs periodic tasks that maintain platform accuracy, compliance, and operational consistency. Tasks run on predictable intervals with tenant‑scoped isolation.

Common scheduled tasks

  • Training expiration checks
  • Certification renewal reminders
  • Readiness recalculation
  • Risk model updates
  • Data retention and purge checks

Operational safeguards

Scheduled tasks run in isolated queues to prevent cross‑tenant interference and ensure predictable performance.

Audit integrity

All scheduled task executions are logged with timestamp, actor (system), and outcome for compliance review.
