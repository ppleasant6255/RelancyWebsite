Tagging Overview

Tags and metadata provide flexible classification for incidents, cases, employees, and locations. Relancy enforces tagging standards to maintain consistency and reporting accuracy.

Tag types

  • Category tags
  • Risk tags
  • Behavioral pattern tags
  • Location‑specific tags
  • Custom tenant tags

Governance rules

  • Tags must be approved by administrators
  • Tags cannot duplicate existing categories
  • Tags follow RBAC visibility rules
  • Tags appear in reports and dashboards

Operational impact

Tags improve search, filtering, routing, and risk analysis across modules.
