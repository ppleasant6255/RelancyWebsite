MFA Overview

Multi‑Factor Authentication (MFA) adds an additional layer of security by requiring users to verify their identity using a second factor. MFA is strongly recommended for all tenants and required for sensitive roles.

Enforcement rules

  • Mandatory MFA for HR and Security roles
  • Optional MFA for employees and managers
  • Configurable tenant‑wide MFA policies
  • Device‑bound session validation

Supported MFA methods

  • Authenticator apps
  • SMS codes
  • Hardware tokens (optional)

Audit integrity

All MFA events—including enrollment, failures, and bypass attempts—are logged for security and compliance review.
